|------|------|

## SURVIVAL::rest 休息规则

| 休息类型 | 恢复 | 要求 |
|----------|------|------|
| 短休（1小时） | 恢复1d6生命 | 安全地点 |
| 长休（8小时） | 恢复全部生命 | 安全地点、充足食物 |
| 完全休息（24小时） | 恢复所有状态 | 安全地点、充足食物和水 |

## SURVIVAL::travel 旅途规则

### 每日行程
- 正常行军：8小时/天，24-32公里
- 强行军：10小时/天，40-48公里（疲劳+1）
- 慢速行军：6小时/天，16-20公里（更安全）

### 补给消耗
- 步行：口粮2银/天
- 骑马：草料1银/天 + 口粮2银/天

### 天气影响

| 天气 | 影响 |
|------|------|
| 晴天 | 正常行军 |
| 小雨 | 行军速度-10% |
| 大雨/暴风雪 | 行军速度-30%，需住宿 |
| 极端天气 | 必须停留 |

## SURVIVAL::hunger 饥饿与脱水

### 饥饿阶段

| 天数 | 状态 | 效果 |
|------|------|------|
| 0-1 | 正常 | 无惩罚 |
| 2 | 轻度饥饿 | -1所有检定 |
| 3 | 中度饥饿 | -1所有检定 |
| 4+ | 严重饥饿 | -2所有检定，虚弱 |

### 脱水阶段

| 天数 | 状态 | 效果 |
|------|------|------|
| 0-1 | 正常 | 无惩罚 |
| 2 | 轻度脱水 | -1所有检定 |
| 3 | 中度脱水 | -2所有检定 |
| 4+ | 严重脱水 | -3所有检定，昏迷风险 |

## SURVIVAL::encumbrance 负重限制

| 负重等级 | 负重上限 | 检定惩罚 |
|----------|----------|----------|
| 轻载 | ≤50磅 | 无 |

## SURVIVAL::shelter 庇护所

### 庇护所类型

| 类型 | 防护效果 | 建造时间 | 材料成本 |
|------|----------|----------|----------|
| 露天 | 无 | 0 | 0 |
| 岩石遮蔽 | 防雨 | 1小时 | 0 |
| 简易帐篷 | 防风防雨 | 2小时 | 5银 |
| 庇护所 | 良好防护 | 8小时 | 20银 |
| 旅馆 | 最佳防护 | - | 5-10银/晚 |

### 庇护所效果
- 有庇护所：休息效果正常
- 无庇护所：长休恢复减半，疲劳+1
