# COMBAT.md — 战斗规则

> **用途**: 战斗判定、攻击伤害、防御、状态效果
> **何时检索**: 进入战斗或需要战斗检定时
> **tags**: [combat, combat, initiative, damage, ac, status]

---

## COMBAT::initiative 先攻顺序

1. 每轮开始时投 `1d20 + 敏捷调整值`
2. 从高到低排序
3. 相同敏捷时，平局双方各投一次

### 回合结构
1. 玩家行动宣言
2. DM 描述结果/要求检定
3. 投掷判定
4. DM 描述结果

## COMBAT::attack 攻击与伤害

### 近战攻击

| 武器 | 伤害 | 速度 | 备注 |
|------|------|------|------|
| 拳头 | d3 | 快 | 无伤害加值 |
| 匕首 | d4 | 快 | 可隐蔽 |
| 手斧 | d6 | 中 | 可投掷 |
| 短剑 | d6 | 中 | 常用武器 |
| 长剑 | d8 | 慢 | 熟练+1 |
| 棍棒 | d4 | 中 | 易于获得 |
| 钉头锤 | d8 | 慢 | 对盔甲+1 |

### 远程攻击

| 武器 | 伤害 | 射程 | 备注 |
|------|------|------|------|
| 投石 | d2 | 短 | 随地可取 |
| 飞刀 | d3 | 中 | 可隐蔽 |
| 短弓 | d6 | 中 | 需要弓 |
| 弩 | d8 | 远 | 慢装填 |

### 伤害公式
```
伤害 = 武器骰 + 力量调整值（近战）/ 敏捷调整值（远程）
```

## COMBAT::defense 防御

### 闪避（AC）

| 状态 | AC |
|------|-----|
| 无甲 | 10 + 敏捷调整值 |
| 皮革甲 | 11 + 敏捷调整值 |
| 锁子甲 | 13 + 敏捷调整值（敏捷-1） |
| 板甲 | 16 + 敏捷调整值（敏捷-2） |

### 格挡
- 需要使用盾牌或副手持武器
- 成功格挡：使一次攻击失效
- 失败：照常受到伤害

## COMBAT::status 状态效果

| 状态 | 效果 | 持续时间 |
|------|------|----------|
| **受伤** | -1攻击/防御 | 治愈前 |
| **重伤** | -2攻击/防御，失血1/轮 | 治愈前 |
| **昏迷** | 无法行动 | 1d6分钟或治愈 |
| **中毒** | 力量/敏捷-2 | 1d4小时或解毒 |
| **震慑** | 无法行动 | 1轮 |
| **恐惧** | 需意志检定否则逃跑 | 1轮或通过 |
| **隐身** | 攻击前敌人无法先攻 | 直到攻击或被发现 |

## COMBAT::environment 环境因素

| 因素 | 影响 |
|------|------|
| **狭窄空间** | 无法夹击，闪避+1 |
| **黑暗/雾** | 远程攻击劣势，察觉-2 |
| **泥泞/不平整** | 移动-1/2 |
| **易燃物** | 火伤害+2 |
| **高处** | 投掷武器+1，近战-1 |
| **民众围观** | 可能引发骚乱/通缉 |

## COMBAT::non_lethal 非致命冲突

### 擒抱
- 力量对抗
- 失败者被压制，无法使用双手

### 推搡
- 力量对抗
- 失败者倒地（需1轮起身）

### 胁迫
- 使用武器威胁
- 目标需意志检定（对抗威胁程度）
