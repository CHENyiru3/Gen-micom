## 建国与核心历史（高层摘要）
- 建立：三族（飞马/独角兽/陆马）联合对抗风之魔（Windigos），终结无尽风暴，形成共同体基础。
- 两公主共治：宇宙公主与月亮公主早期共同统治，和平图景被“混乱与不和谐”象征者无序（draconequus）挑战。
- 梦魇之月：月亮公主的黑暗形态导致“永恒黑夜”威胁；和谐之元将其流放千年；后由“和谐之元使者群体”击败并复归。
- 幻形族与首都事件：幻形族以伪装与情感为资源尝试入侵，曾对坎特洛特造成重大危机。
- 水晶帝国：北境王国曾随统治者森布拉的威胁而消失千年，后重现并受水晶之心守护。

## 关键地点速记（详细条目见 locations/**）
- 小马谷（Ponyville）：农业与日常生活的中心舞台；友谊实践密度高，适合“关系推进”主轴。
- 坎特洛特（Canterlot）：王都与政治/学术/贵族重心，适合“身份与制度”主题分线。
- 永恒自由森林（Everfree Forest）：自然与魔法异常区域，适合“未知与试炼”分线。
- 水晶帝国（Crystal Empire）：北境寒地王国，适合“旧咒与复兴”分线。
