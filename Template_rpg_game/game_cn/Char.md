## 5) Death and Resurrection

### 5.1 Falling to 0 HP
- Unconscious
- Must make death saving throws

### 5.2 Death Saving Throws
- Roll 1d20
- 10+ = success
- < 10 = failure
- 3 successes = stabilized
- 3 failures = death
