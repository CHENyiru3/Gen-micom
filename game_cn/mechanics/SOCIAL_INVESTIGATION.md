# SOCIAL_INVESTIGATION.md — 社交与调查规则

> **用途**: 社交检定、调查行动、洞察判断
> **何时检索**: 需要社交互动或调查时
> **tags**: [social, investigation, persuasion, deception, insight]

---

## SOCIAL::checks 社交检定

### 基础公式
```
1d20 + 调整值 vs 目标DC
```

### 调整值来源

| 因素 | 调整值 |
|------|--------|
| 魅力基础 | +0~+3 |
| 伪装身份加成 | +1~+3 |
| 贿赂/礼物 | +2~+5 |
| 知识优势 | +1~+3 |
| 不利条件（时间紧迫等） | -1~-3 |

## SOCIAL::types 社交类型

### 说服（Persuasion）
- 用于：请求帮助、提供合理论点、谈判
- DC基础：10（容易）~20（困难）

### 欺骗（Deception）
- 用于：说谎、伪装身份、误导
- DC基础：12（简单谎言）~18（复杂谎言）
- 目标可能进行"洞察"检定

### 恐吓（Intimidation）
- 用于：威胁、施压、逼供
- DC基础：12（胆小目标）~20（硬汉）
- 失败可能导致敌对

### 表演（Performance）
- 用于：吸引注意、演讲、伪装成特定身份
- DC基础：10（普通）~25（专家）

## SOCIAL::insight 洞察检定

### 目标DC
```
DC = 8 + 目标"洞察"技能 + 等级/经验调整
```

### 洞察时机
- 玩家说谎后
- NPC怀疑玩家意图
- 涉及重要秘密的对话

### 洞察结果

| 差距 | 结果 |
|------|------|
| ≥10 | 完全识破 |
| 5-9 | 怀疑，进一步追问 |
| 0-4 | 没有明显怀疑 |
| <0 | 相信玩家（即使在说谎） |

## INVESTIGATION::time 时间压力

| 情况 | 时间压力 |
|------|----------|
| 目击者即将离开 | -2洞察DC，必须快速 |
| 证据可能被销毁 | 搜索DC+2 |
| 宵禁前必须完成 | 时间限制明确 |
| NPC有其他急事 | 减少社交时间 |

---

## INVESTIGATION::risk 社交风险

| 行动 | 风险 |
|------|------|
| 在公共场合说谎 | 被听到，Rumor+1 |
| 威胁有背景的NPC | 可能被报复，Heat+1 |
| 伪装身份被识破 | 可能被抓，Heat+2 |
| 行贿被揭发 | 通缉风险 |

## INTERROGATION::rules 审讯规则

### 目标意志检定

| 目标 | DC基础 |
|------|--------|
| 普通人 | 10 |
| 有训练的士兵 | 15 |
| 意志坚定者 | 20 |
| 死士 | 无法审讯 |

### 审讯方法

| 方法 | 调整值 | 风险 |
|------|--------|------|
| 物理威胁 | +2 | Heat+1 |
| 金钱利诱 | +2 | 需支付 |
| 欺骗 | +1 | 识破则失败 |
| 心理施压 | +1 | 时间+ |

### 后果
- 成功：获得信息
- 失败：目标沉默或说谎
- 严重失败：目标反抗或死亡
