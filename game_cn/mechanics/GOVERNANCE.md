## GOV::settlement 回合结算

### 收入计算
```
总收入 = 农业产出 + 商业利润 + 税收 - 维护成本
```

### 支出项目

| 项目 | 费用 |
|------|------|
| 军队月薪 | 按人数×训练度计算 |
| 官员俸禄 | 按职位计算 |
| 建筑维护 | 按建筑数量计算 |
| 突发事件 | 疾病、灾害、战争赔偿 |

### 净收益
```
净收益 = 总收入 - 总支出
正数：积累财富，可用于投资/储蓄
负数：消耗储备，忠诚度下降风险
```

## GOV::followers 追随者系统

### 获取方式

| 方式 | 成功率 | 风险 |
|------|--------|------|
| 招募流浪者 | 高 | 忠诚度低 |
| 雇佣专业人士 | 中 | 费用高 |
| 征服获得战俘 | 低 | 需要转化 |
| 吸引难民 | 高 | 管理难度大 |
| 拯救村庄 | 中 | 需保护承诺 |

### 日常管理

- 每月消耗食物 = 追随者数量×2单位
- 食物不足 → 忠诚度-1/月，逃亡概率+10%
- 定期奖励 → 忠诚度+1

### 忠诚度事件

| 事件 | 忠诚度变化 |
|------|-----------|
| 战斗胜利 | +1~2 |
| 战斗失败 | -1~3 |
| 发放奖金 | +1~2 |
| 克扣物资 | -2~5 |
| 主公战死 | 追随者溃散 |

## GOV::military_units 军事单位管理

### 单位创建
```
招募成本 = 基础装备 + 训练费 + 安置费
训练时间 = 基础训练 × (10 - 训练度) / 10
```

### 单位升级

| 升级 | 条件 | 效果 |
|------|------|------|
| 民兵→正规军 | 训练30天 + 实战1次 | 战斗力+2 |
| 正规军→精锐 | 训练60天 + 军费X | 战斗力+3 |
| 精锐→亲卫 | 特殊任务 + 主公认可 | 专属忠诚 |

### 标准配置（100人）
- 20% 弓箭手（远程）
- 30% 长枪兵（防御）
- 30% 剑士（突击）
- 20% 骑兵（机动）

### 特殊单位

| 单位 | 特点 |
|------|------|
| 重甲骑士 | 训练成本3倍，维护成本2倍 |
| 游侠斥候 | 侦察+2，战斗-1 |
| 工程兵 | 攻城效率+3，战斗-2 |

## GOV::buildings 建筑管理

### 建筑类型

| 建筑 | 功能 | 建造时间 | 建造费用 |
|------|------|----------|----------|
| 农舍 | 容纳10人 | 10天 | 银币50 |
| 仓库 | 储存100单位 | 15天 | 银币100 |
| 磨坊 | 粮食加工 | 20天 | 银币150 |
| 铁匠铺 | 武器修理 | 25天 | 银币200 |
| 兵营 | 士兵训练 | 30天 | 银币300 |
| 要塞 | 防御+5 | 60天 | 金币50 |
| 城墙 | 防御+10 | 90天 | 金币100 |
| 港口 | 船只停靠 | 45天 | 金币80 |

### 建议发展顺序
1. 农舍（容纳人口）
2. 仓库（储存资源）
3. 磨坊（提升产出）
4. 兵营（训练军队）
5. 铁匠铺（装备维护）
6. 要塞/城墙（防御）
7. 港口（贸易扩展）
