## 0. 城市气质

内伯尔海姆的真实样子不是大教堂尖顶，而是：

- **泥泞、尿液、皮革腐臭、烟灰与啤酒泡沫**混在一起
- 祈祷词和咒骂声同样常见
- 人们相信上帝，也相信账本与匕首
- "正义"白天在市政厅，夜里在树林的绳索上

---

## 1. 城市分区

### 1.1 外层棚户与难民带：饥饿的围城

| 特征 | 描述 |
|------|------|
| **位置** | 城墙外侧 |
| **景观** | 棚屋、烂木板与破布；夜里火光像萤火，白天臭味像雾 |
| **信仰** | "面包教派"：有人用面包屑占卜，声称能看见冬天会死多少人 |
| **黑市** | 偷来的盐、病死牲口肉、假药、假通行证、伪造圣符 |
| **执法** | 城卫偶尔来"清理"，更多时候来收钱 |

**常见冲突**：抢水、抢柴、偷粮、传染病隔离、替罪羊暴动

---

### 1.2 行会街与工坊带：噪音、手艺与算计

| 特征 | 描述 |
|------|------|
| **声音** | 锤铁声、织机声、屠宰声连成一条"工业祷告" |
| **行会权力** | 不仅管手艺，也管婚姻、葬礼、民兵、投票；你得入会才算"人" |
| **学徒处境** | 挨打是家常便饭，逃跑会被贴告示追捕 |
| **暴力形式** | "账本暴力"：债务催收像猎犬 |

**可用地点**：
- 染坊（强烈气味、易投毒）
- 屠宰场（血与污水）
- 铁匠铺（武器与谣言）
- 织工塔楼（政治秘密会议）

---

### 1.3 市集与酒馆：流言的发动机

| 特征 | 描述 |
|------|------|
| **安全** | 啤酒比水安全，酒馆比教堂更像公共议会 |
| **活动** | 赌局、斗殴、卖淫、招工、密谈在同一张桌子附近发生 |
| **情报** | "讲故事的人"能卖情报：每个故事都是真假掺半，越恐怖越值钱 |
| **骗局** | 假圣物、假赎罪券、假炼金、假异端名单（用来勒索） |

**招牌景观**：
- "赎罪唱诗班"在酒馆门口唱歌讨钱；唱到后半段会变成侮辱主教的俚语
- "瘸腿的先知"兜售末日日期，日期每周变一次，但总有人信

---

### 1.4 犹太巷与账房：文字的刀

| 特征 | 描述 |
|------|------|
| **门禁** | 铁门让这里像一座"夜间监狱" |
| **功能** | 城市最高效的医疗与借贷节点 |
| **气味** | 药香、墨水味、羊皮纸味 |
| **危险** | 不是匕首，是：**账目、契约、证词** |
| **风险** | 瘟疫或粮价崩溃时，暴民会把"复杂"当成罪 |

> **DM提示**：不要把这里写成单一刻板形象；写出"被需要又被憎恨"的结构性矛盾。

---

### 1.5 大教堂区与修会：光与腐朽同居

| 特征 | 描述 |
|------|------|
| **景观** | 哥特尖顶刺向天空，但脚手架与未完工的石材堆更像伤口 |
| **权力** | 掌握慈善、抄写、审判、收容 |
| **阴暗面** | 告密网、赦免与禁令 |
| **告解室** | 能听到最肮脏的秘密 |
| **祭坛后** | 能看到最干净的谎言 |
| **圣所权** | 法律缓冲，也是罪犯与政治犯的谈判桌 |

---

### 1.6 刽子手桥与城外边缘：被诅咒的必要之恶

| 特征 | 描述 |
|------|------|
| **刽子手** | 住得远离人群；人们需要他，却不愿在街上和他对视 |
| **交易** | 草药、止痛粉、堕胎药、毒鼠饵，也收购"别人不敢收的东西" |
| **夜间交易** | 尸体、骨头、赎金、假身份、黑话符号 |
| **城外刑场** | 椴树是"社会的终点站"，树干上的匕首能让你一夜成名（或永远闭嘴） |

## 2. 不良之地目录

可直接用作遭遇表或任务地点：

| 地点 | 风险 | 机会 |
|------|------|------|
| **廉价妓院** | 债务绑人；保护费交给城卫或雇佣兵 | 情报、线索、特殊关系 |
| **赌坊与斗犬场** | 赌债=暴力合同 | 斗犬师知道很多秘密 |
| **鸦片/草粉屋**（睡梦粉/修士之尘） | 成瘾、公开谴责 | 止痛、信息、精神冒险 |
| **下水道工与粪坑队** | 城市地下地图的代价 | 带你去你不该去的地方 |
| **拐卖与勒索链** | 最常见的猎物 | 拯救任务、敌人网络 |

**DM规则**：不良之地不是"奖励关卡"——提供资源与情报，但必带社会后果（Rumor/Heat、信用、敌人）。

## 3. 黑话与符号系统

### 3.1 黑话三原则（Rotwelsch）

- 词汇短、粗、带动物比喻（"狼口=收税官""白手=没钱""黑面包=脏活"）
- 每个团体都有自己的"暗号句式"
- 黑话是阶层武器：听不懂的人就会付更多钱

### 3.2 门框刻痕两套系统

| 系统 | 描述 | 来源 |
|------|------|------|
| **流民标记** | 此处好施舍/有恶犬/警察凶/可借宿 | 白昼史 |
| **雾印（Mist Marks）** | 形似流民刻痕但更精细，指向"裂纹点"或"阈值门" | 迷雾史 |

## 4. 市井日常剧情引擎

这些不是背景装饰，是每回合都能用的推动器：

| # | 剧情引擎 | 触发条件 | 可能后果 |
|---|----------|----------|----------|
| 1 | **面包队列冲突** | 粮价上升/断供 | 抢粮、谣言、暴力、收买 |
| 2 | **城卫敲诈** | 任意街头行动 | 拒付→搜身/栽赃 |
| 3 | **雇佣兵夜巡** | 夜间外出 | "保护费"勒索、暴力 |
| 4 | **工坊事故** | 探索行会区 | 火灾/化学染料毒/塌方；救人影响 Grace/Debt |
| 5 | **替罪羊时刻** | 怪异/瘟疫后 | 人群要找"原因"，最弱者最先被指 |
| 6 | **告解室交易** | 教堂区域 | 秘密换赦免，赦免换钱，钱换沉默 |
| 7 | **黑市圣物** | 探索黑市 | 真假难辨；买错会涨 Heat；买对会引来追杀 |
