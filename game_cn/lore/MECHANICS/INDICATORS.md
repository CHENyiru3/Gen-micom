## IND::001 四大全局指标

| 指标 | 范围 | 含义 | 高值效果 | 低值效果 |
|------|------|------|----------|----------|
| **恩典 Grace** | 0-10 | 神圣秩序可用性 | 祝圣有效、圣所坚固 | 祷告沉默、圣物失效、异象频发 |
| **罪债 Debt** | 0-10 | 集体赎罪压力 | 怪物/梦疫/诱惑增多；社会更暴烈 | 相对平静、迷雾退缩 |
| **谣言 Rumor** | 0-3 | 城市对怪异/施法的议论程度 | 调查力量必介入 | 怪异可隐藏 |
| **关注 Heat** | 0-3 | 城卫/修会/裁判/菲姆法庭的注意度 | 直接干预、追捕、审判 | 自由行动空间 |

## IND::002 指标核心法则

### 2.1 指标互动规则

```
 Debt+1 / Heat+1 ← 捷径暴力/滥用魔法解决关键问题
 Debt-1 / Grace+1 ← 承担代价救人/拒绝诱惑/保护无辜
 Rumor≥2 ← 必引来某类调查/审判力量介入
```

### 2.2 恩典阈值效果

| Grace 值 | 效果 |
|----------|------|
| 8-10 | 祝圣有效+1，圣所坚固，治愈神术+1 |
| 5-7 | 正常圣所保护，治愈神术正常 |
| 3-4 | 圣所可能"漏风"，治愈神术-1 |
| 1-2 | 祷告常无回应，圣物部分失效 |
| 0 | 恩典断绝，怪异可侵入任何圣所 |

### 2.3 罪债阈值效果

| Debt 值 | 效果 |
|---------|------|
| 8-10 | 梦疫高发，怪物刷新+2，街头暴力激增 |
| 5-7 | 中等怪物刷新，诱惑增多 |
| 3-4 | 轻度怪物刷新，偶发诱惑 |
| 1-2 | 相对平静，迷雾退缩 |
| 0 | 迷雾场最弱，怪异几乎不出现 |

### 2.4 谣言阈值效果

| Rumor 值 | 效果 |
|----------|------|
| 3 | 全城皆知，调查力量必介入，多方势力关注 |
| 2 | 城区流传，调查力量可能介入 |
| 1 | 小范围耳语，需特定触发才升级 |
| 0 | 无议论，怪异可隐藏 |

### 2.5 关注阈值效果

| Heat 值 | 效果 |
|---------|------|
| 3 | 追捕中，指定力量介入，可能包围/伏击 |
| 2 | 被标记，调查者主动搜寻 |
| 1 | 被注意，特定势力会核实身份 |
| 0 | 无关注，行动自由 |

## IND::004 势力介入阈值

| 触发条件 | 介入势力 | 介入方式 |
|----------|----------|----------|
| Rumor≥2 + Heat≥1 | 城卫/城市民兵 | 巡逻加强、宵禁、排查 |
| Rumor≥2 + 施法相关 | 修会调查员 | 秘密调查、传唤、审判 |
| Rumor≥2 + 怪物目击 | 菲姆法庭 | 秘密处决、清除污染 |
| Heat≥2 + 贵族相关 | 帝国/法兰西势力 | 外交压力、通缉令 |
| Debt≥7 + Grace≤3 | 多重怪异事件 | 教会净化的极端手段 |
| Rumor=3 + Heat=3 | 多方势力介入 | 复杂冲突、战役级事件 |

```

```
