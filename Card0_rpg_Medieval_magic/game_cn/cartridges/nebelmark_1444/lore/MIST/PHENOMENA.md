## 0. 渗出原则

怪物先以"传闻/象征/个案"出现，等阈值跨越后才更常态化。不需要一开始叫成 D&D 标准名：

- 天罚、恶兆、异端之影、圣徒的反面
- 白昼史解释：异端巫术、神罚预兆、瘟疫症状
- 迷雾史真相：真实存在的异想生物

## 1. 异想生物目录

### 1.1 赎罪鸟（The Penitent Lark）

| 属性 | 描述 |
|------|------|
| **外形** | 灰白小鸟，胸口像被烙过十字，叫声像儿童合唱又像哽咽 |
| **行为** | 出现在濒死者附近；若有人在旁边撒谎或伪装忏悔，它会突然沉默并留下灰尘 |
| **宗教色彩** | 被认为是"恩典撤回时的信号" |
| **玩法用途** | 揭露谎言的线索工具 |
| **后果** | 会提高 Rumor（"你们在和天罚打交道"） |

---

### 1.2 告解蛞蝓（Confessional Slug）

| 属性 | 描述 |
|------|------|
| **外形** | 湿亮的黑色蛞蝓，背部有细小纹路像祷文 |
| **行为** | 在告解室、修道院地窖、潮湿书库出现；爬过的地方会留下让人"想说真话"的黏液 |
| **异端色彩** | 修会称其为恶魔伎俩；异端小团体却把它当成"去伪存真"的圣迹 |
| **危险** | 接触过量会导致精神崩溃或"自白上瘾" |

---

### 1.3 圣痕乞丐（Stigmata Mendicant）

| 属性 | 描述 |
|------|------|
| **外形** | 像人，但伤口在手足处反复渗血；衣物常干净得不合理 |
| **行为** | 在市集中讨钱，不要面包，只要"名字"或"一句誓言" |
| **争议** | 有人认作圣徒，有人认作伪装的巫师 |
| **玩法钩子** | 给了"名字/誓言"的人，会被某种力量"记住"（Heat 或 Debt 触发器） |

---

### 1.4 七灯烛童（Candleborne Children）

| 属性 | 描述 |
|------|------|
| **外形** | 一群孩子般的影子，头顶像有七点烛火；靠近时闻到蜂蜡与冷烟 |
| **行为** | 在十二夜或雾夜出现，带路去"你最想见的人"的地方 |
| **真相** | 它们不是带路，而是带你去"你最愿意付代价的地点" |
| **危险** | 这是诱惑试炼装置，常与 Debt 增长绑定 |

---

### 1.5 盐灰修士（Ash-Salt Friar）

| 属性 | 描述 |
|------|------|
| **外形** | 披斗篷的修士轮廓，衣下是盐与灰混成的粉末；走过会留下细白脚印 |
| **行为** | 在疫病与饥荒高发处出现；它会把盐撒在尸体上，尸体随即变"更安静" |
| **宗教色彩** | 有人视作仁慈天使，有人认为它在"偷走灵魂" |
| **玩法用途** | 能暂时压制不死征兆，但代价可能是 Grace 下降（"恩典被替换为秩序"） |

---

### 1.6 雾庭辩者（Mist Court Advocate）

| 属性 | 描述 |
|------|------|
| **外形** | 穿黑袍的"律师/书记"，脸被雾遮住，手持羊皮纸卷 |
| **行为** | 在债务纠纷、赎金谈判、私战宣言现场出现；它提出"完美合法"的条款 |
| **异端色彩** | 它让法律本身变成诅咒；签了就很难脱身 |
| **玩法钩子** | 把白昼史与迷雾史粘合的怪物——从"合同"进入人间 |

## 2. 怪异现象目录

### 2.1 裂纹期常见"症状"

> 这些现象先以"传闻/个案"出现，逐步变为可重复事件。

| 现象 | 描述 | 白昼史解释 |
|------|------|------------|
| **梦疫** | 同一晚多人做同一梦，醒来手心有灰 | 集体癔症、瘟疫前兆 |
| **雾印** | 门框出现细小刻痕，指向某条路或某个地窖 | 流民标记 |
| **漏圣所** | 教堂内听见"风在祷告里穿行"，蜡烛无故熄灭 | 建筑老化、通风 |
| **错位账本** | 交易记录被改写，债务突然"合法化" | 伪造、欺诈 |
| **树上的匕首** | SSGG 作为"合法处决"标记出现，尸体表情像在倾听 | 秘密法庭处决 |
| **狂猎残影** | 十二夜期间天空有马蹄声；猎犬吠叫来自云层 | 暴风雪、狼嚎 |

### 2.2 迷雾史增补

| 现象 | 迷雾史解释 |
|------|------------|
| **梦疫** | 集体意识与迷雾场的共振 |
| **雾印** | 指向"裂纹点"或"阈值门"（地下城入口） |
| **漏圣所** | 恩典边界"漏风"，怪异可越界 |
| **错位账本** | 迷雾对"契约"概念的物理篡改 |
| **树上的匕首** | 菲姆法庭在"筛选迷雾污染" |
| **狂猎残影** | 狂猎是真实存在的跨维度狩猎 |

## 3. 宗教与异端团体

### 3.1 灯下读经会（地下改革小组）

| 项目 | 内容 |
|------|------|
| **表面** | 识字工匠夜间读经 |
| **实质** | 传播白话经文与反什一税思想 |
| **与怪异关系** | 认为怪异是"上帝对腐败的审判"，不怕，甚至拥抱 |
| **可能线索** | 与雾印路径相连 |

### 3.2 圣约净化团（修会极端派）

| 项目 | 内容 |
|------|------|
| **目标** | 恢复秩序，宁可牺牲无辜也要"净化" |
| **手段** | 告密网、临时审讯、封锁街区、焚书、强制忏悔 |
| **与怪异关系** | 把异想生物当证据，借此夺权或清洗派系 |

### 3.3 灰盐行脚医（边缘知识者网络）

| 项目 | 内容 |
|------|------|
| **成员** | 刽子手、草药师、犹太医生、退役军医、异端"巴尔布" |
| **目标** | 救人、活下去、保持知识流动 |
| **与怪异关系** | 不崇拜也不否认，只把它当病理与环境变量处理 |

## 4. DM 使用规则

### 怪异出现原则
- 每次怪异目击都应提升城市紧张度
- 至少改变一个指标（Rumor/Heat/Grace/Debt）
- 怪异出现要带"宗教解释权之争"：修会说是恶魔，异端说是启示

### 触发条件

| 指标状态 | 可能触发的怪异 |
|----------|----------------|
| Grace ≤ 3 | 赎罪鸟、漏圣所、盐灰修士 |
| Debt ≥ 7 | 告解蛞蝓、圣痕乞丐、雾庭辩者 |
| Rumor ≥ 2 | 七灯烛童、狂猎残影 |
| Heat ≥ 2 | 圣约净化团行动、灯下读经会渗透 |
